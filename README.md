# Flight game

Open `index.html` in a modern desktop or mobile browser. Press Space, any arrow key, W, click, or tap to fly.

The course starts forgiving, then the pipe opening and pipe width gradually narrow while speed increases. It changes to a new mountain scene at 25 points and ends at 99 points.

To turn on the shared online leaderboard, follow [SUPABASE_SETUP.md](SUPABASE_SETUP.md).
